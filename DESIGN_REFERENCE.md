# Design Reference - PagesJaunes Style for YellowHalal

## HEADER
- Fond blanc solide, pas de glassmorphism
- Logo YellowHalal à gauche
- Navigation simple texte : Explorer, Professionnels, Blog, Aide
- Bouton CTA à droite : "Référencer mon entreprise" (jaune)
- Hauteur ~60px, border-bottom gris léger
- PAS de barre de recherche dans le header

## HERO
- Fond jaune vif #FFC200, pleine largeur
- Titre centré : "Trouver des services 100% Halal près de chez vous"
- Sous-titre : texte descriptif court
- BARRE DE RECHERCHE AU MILIEU : grande, blanche, shadow, 2 champs côte à côte
  - Champ 1 : "Que recherchez-vous ?" avec icône loupe
  - Champ 2 : "Où ? (ville, code postal)" avec icône pin
  - Bouton recherche jaune/noir à droite
- Tags rapides sous la barre : "Restaurant halal", "Boucherie", "Mosquée"...

## CATEGORIES
- Fond blanc
- Rangée horizontale d'icônes RONDES (cercles avec fond coloré léger)
- Icône au centre du cercle, label texte en dessous
- 8 catégories sur une ligne, scrollable sur mobile
- Style PagesJaunes : simple, propre, pas de cartes

## BANNIÈRES PUB
- Grandes images de fond avec overlay sombre
- Texte blanc en overlay : titre + description + bouton CTA
- Coins arrondis, pleine largeur dans le container
- Label "Publicité" discret en haut

## CARTES ENTREPRISES (Top 3)
- Fond blanc, border gris léger, border-radius 12px
- Layout horizontal : image à gauche, infos à droite
- Nom en gras, catégorie en gris
- Étoiles jaunes pour le rating
- Badge vert "Certifié Halal"
- Boutons : "Afficher le N°" (bleu/noir), "Itinéraire", "Écrire un avis"
- Adresse + distance

## ARTICLES
- Carrousel horizontal ou grille 3 colonnes
- Photo en haut (border-radius), date + catégorie, titre en gras
- Style blog classique, pas de gradients CSS

## CTA PROS
- Fond noir ou gris très foncé
- Texte blanc, titre accrocheur
- Bouton CTA jaune "Inscrire mon établissement"
- Stats ou avantages listés

## FOOTER
- Fond gris foncé (#1C1C1E)
- 4-5 colonnes de liens blancs
- Logo en haut à gauche
- Copyright en bas
- Language switcher

## PRINCIPES GÉNÉRAUX
- Fond de page : blanc ou gris très clair (#F5F5F5)
- Pas de gradients CSS fantaisistes
- Pas d'animations complexes
- Espacement généreux entre sections
- Typographie claire et lisible
- Style UTILITAIRE, pas design startup
- Couleurs : jaune #FFC200, noir #0A0A0A, gris #F5F5F5, blanc
